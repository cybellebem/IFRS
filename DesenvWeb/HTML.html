HTML

HyperText Markup Language

Linguagem de marcações/tags para estruturar conteúdos na web.

Padronizada inicialmente pela IETF – RFC1866 (HTML 2.0, 1995).

Atualmente mantida pelo W3C e WHATWG (HTML5).

Tipos de Tags

  • Abertura e Fechamento (compostas): <p></p>
  • Vazias (self-closing): <img>

Estrutura Básica de um Documento

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Título</title>
  </head>
  <body>
    <h1>Olá Mundo!</h1>
  </body>
</html>

Portlets

Portlet é um componente web modular.

Funciona como um miniaplicativo dentro de um portal.

Similar a widgets e cards.

Um portal reúne vários portlets, cada um com sua própria lógica, formulários e estilos.

Atributo aria-label

Pertence ao padrão WAI-ARIA.

Define um rótulo textual para elementos e ícones.
