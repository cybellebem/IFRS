CSS3

Cascading Style Sheets

Criado para separar o conteúdo da apresentação visual em documentos HTML e XML.

Especificidade

  1. !important
  2. Estilos inline: diretamente no elemento (ex: style="color:red;")
  3. IDs (ex: #identificador)
  4. Classes (.classe), Pseudo-classes (:hover), atributos (type="text")
  5. Elementos (div,p)
  6. Universal (*)
  7. Herança: não tem Especificidade própria

Tipos de Declaração CSS

  • Inline: <p style="color:red">
  • Interno: <style></style>
  • Externo: <link rel="stylesheet" href="mobile.css" media="(max-width: 600px)">

Seletores

Básicos

  • #id
  • .classe
  • elemento
  • * (universal)

Combinados

  • A, B → h1, p { }
  • A B → div p { }
  • A > B → div > p { }
  • A + B → h1 + p { }
  • A ~ B → h1 ~ p { }

Pseudo-classes (:)

  • :hover
  • :focus
  • :active
  • :nth-child(n)
  • :first-child
  • :last-child
  • :not(selector)
  • :checked
  • :disabled / :enabled
  • :visited

Pseudo-elementos (::)

  • ::before / ::after
  • ::first-line
  • ::first-letter
  • ::selection

Seletores por Atributo

  • [attr]
  • [attr=value]
  • [attr~=value]
  • [attr^=value]
  • [attr$=value]
  • [attr*=value]

Modificadores

  • [attr=value i] → ignora maiúsculas/minúsculas
  • [attr=value s] → sensível a maiúsculas/minúsculas

At-rules (@)

  • @import
  • @media
  • @font-face
  • @keyframes
  • @supports
  • @page

Display

  • block → ocupa toda largura e quebra linha
  • inline → ocupa apenas o conteúdo
  • inline-block

    • Display: block

    Block 1
    Block 2
    Block 3

    ✔ Quebra linha automaticamente
    ✔ Ocupa toda largura disponível
    ✔ Aceita width e height

    Display: inline

    Inline 1 Inline 2 Inline 3

    ✔ Não quebra linha
    ✖ Não aceita width e height
    ✔ Ocupa apenas o tamanho do conteúdo

    Display: inline-block

    Inline-Block 1
    Inline-Block 2
    Inline-Block 3

    ✔ Não quebra linha
    ✔ Aceita width e height
    ✔ Permite controlar tamanho mantendo comportamento inline

  • flex → container flexível
  • float → alinhamento clássico horizontal

Cores

Formatos

  • Hexadecimal → #ff0000
  • RGB → rgb(255,0,0)
  • RGBA → rgba(255,0,0,0.5)
  • HSL → hsl(0,100%,50%)

Gradientes

  • Linear → linear-gradient(to right, red, yellow)
  • Radial → radial-gradient(circle, red, yellow)

Box Model

Define espaço e tamanho dos elementos.

box-sizing

  • content-box → padrão
  • border-box → inclui padding e border

Reset CSS e Normalize

  • Reset CSS: remove estilos padrões
  • Normalize.css: mantém estilos úteis e corrige inconsistências
