JSON

  • Javascript Object Notation
  • Alternativa ao XML
  • Leve de armazenar e transportar dados
  • Derivado do JS, mas é independente dele, sendo usado em muitas linguagens de programação através de bibliotecas.
  • Usado para enviar dados entre um servidor e uma aplicação web, ou entre sistemas computacionais principalmente em APIs e configurações web

  • array: []
  • Valor para ausência de informação: null;
